Description:
    This is a website for a shopping website called Taobao.
    It conatins only one page:
    1. A nav bar on the top, including search field and a bunch of buttons
    2. Button login pops a offcanvas to show the login form.
    3. Carousel shows pictures.
    4. Two cards gallery shows in different styles, present products we provide.
    5. Detail button in cards. And for the second cards gallery, we used accirdion. 
    6. A footer contains future functions and claim copyright.
